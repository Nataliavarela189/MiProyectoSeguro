# 🚀 Aplicación Fullstack (.NET 8 + Angular 16)

Aplicación fullstack desarrollada como prueba técnica, implementando buenas prácticas modernas de arquitectura backend y frontend.

## 🧩 Descripción

El proyecto consiste en una aplicación web completa con:

* Backend en **.NET 8 (C#)**
* Frontend en **Angular 16+**
* Persistencia con **Entity Framework + SQLite**

Se enfoca en una arquitectura limpia, escalable y mantenible.

---

## 🏗️ Arquitectura

El backend implementa:

* **CQRS (Command Query Responsibility Segregation)**
* **Patrón Repository**
* Separación clara de responsabilidades
* Manejo estructurado de lógica de negocio

---

## 🔐 Seguridad

* Autenticación de usuarios
* Autorización basada en roles/permisos
* Protección de endpoints

---

## ✅ Validaciones y Calidad

* Validaciones de datos en backend
* Manejo de errores controlado
* Logging para trazabilidad
* Pruebas unitarias con **NUnit**

---

## 💻 Tecnologías

### Backend

* .NET 8 (C#)
* Entity Framework Core
* SQLite
* CQRS
* Repository Pattern

### Frontend

* Angular 16+
* TypeScript
* HTML / CSS

### Testing

* NUnit

---

## ▶️ Ejecución del proyecto

### Backend

```bash
cd backend
dotnet restore
dotnet run
```

### Frontend

```bash
cd frontend
npm install
ng serve
```
